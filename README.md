# nice-simple-captcha

> Simple Captcha

## Install

```bash
npm install --save nice-simple-captcha
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'nice-simple-captcha'
import 'nice-simple-captcha/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
